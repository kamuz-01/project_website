MUITO PRAZER, Meu nome é Karli De Jesús Munoz Manzano.

Nasci na Venezuela e atualmente estou cursando a sexta fase do curso superior de Aálise e Desenvolvimento de Sistemas e meu objetivo é me tornar desenvolvedor web com foco em aplicações modernas usando tecnologias modernas como o framework Django e Spring Boot para back-end e React, Vue.js ou o tradicional HTML5, CSS3 e JavaScript para front-end.

Minha jornada com programação começou em 2023, quando descobri minha paixão por criar soluções digitais. Desde então, venho me dedicando ao estudo de desenvolvimento web, com ênfase em Python, Django Java e Spring Boot. Tenho experiência com HTML, CSS, JavaScript, Python, Django, Java, Spring Boot, JQuery e Bootstrap, além de conhecimentos em Git e bancos de dados como MySQL, MariaDB, PostgreSQL e SQLite.

Já desenvolvi projetos para gerenciar os processos internos de um hotel, e estou constantemente buscando novos desafios para crescer na área. Quando não estou programando, gosto de aprender sobre design, ler sobre filosifia, religião, política, atualidades tecnológicas, além de explorar novas culturas.
